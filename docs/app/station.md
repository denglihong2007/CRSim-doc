1.在车站数据库中使用"+"按钮新建一个车站

![](img\station\1.png)

2.点击车站名，新建候车室，检票口，站台

![](img\station\2.png)

::: tip 提示

你可以在检票口或站台中输入1A-28B，1-28这样的格式来批量创建候检票口或站台。

:::

3.点击车站车次的"+"，添加车次

![](img\station\3.png)

::: tip 提示
你可以通过最靠右的按钮从其他地方导入车次数据，一般使用联网获取

![](img\station\4.png)

联网获取速度可能较慢，请耐心等待。
:::

4.不要忘记保存配置

![](img\station\5.png)

::: info 注意
联网获取的站台数据是随机的，不反应真实情况
:::
